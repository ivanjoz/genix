## DESCRIPTION::ES
Punto de venta (POS). Vista Ventas: buscar productos en stock por nombre o número de serie, agregar al carrito con cantidad, seleccionar caja y almacén, asignar cliente (existente o registrar uno nuevo), y generar la orden de venta. Vista Configuración: activar o desactivar el flujo separado de pago/entrega.

## DESCRIPTION::EN
Point of sale (POS). Sales view: search in-stock products by name or serial number, add to cart with quantity, select cash register and warehouse, assign a customer (existing or register a new one), and generate the sale order. Configuration view: enable or disable the separate payment/delivery flow.
