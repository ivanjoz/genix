## DESCRIPTION::ES
Gestión de perfiles de acceso. Crear y editar perfiles con nombre y descripción. Asignar permisos granulares (ver, editar, etc.) por módulo y recurso a cada perfil. Visualizar todos los accesos del perfil seleccionado agrupados por módulo.

## DESCRIPTION::EN
Access profile management. Create and edit profiles with name and description. Assign granular permissions (view, edit, etc.) per module and resource to each profile. View all permissions of the selected profile grouped by module.
